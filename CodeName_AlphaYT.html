YT Downloader

  • View On Github (v.1.3)

    • About:

    A Python based UI Application capable to download any Youtube Video in Best Available Audio and Vido format.

    The Application includes 3 buttons. One being the button to change the download destination. Second being the button to open log file and the last being the option to view the Read me file.

    When the URL is put in the URL entry box, the program checks it for being valid and then tells the user if it is valid or not. If it is valid it further proceeds to asking user about the format they want to download it. It would either be a MP4 or MP3/WEBM . Afterwards it keeps track of the download process and tells the user of it.

    The Plus Point of using YT Downloader is that it is also capable of downloading your playlists with ease, given they are public
